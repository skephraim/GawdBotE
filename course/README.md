# GawdBotE Mini Course
### Build a Self-Improving AI Assistant from Scratch

This course teaches you everything used to build GawdBotE — from first principles to production deployment. Every lesson uses real code from the project so you can read, run, and modify it as you learn.

---

## Who This Is For

- Developers who want to understand how AI agents actually work (not just API calls)
- People who want to build their own AI assistant
- Anyone who wants to teach this material to others

**Prerequisites:** Basic Python. You don't need to know anything about AI.

## How to Use This Course

Each lesson has:
- **The concept** explained plainly
- **The real code** from GawdBotE with line-by-line commentary
- **A mini exercise** to solidify understanding
- **Key takeaways** you can put on a slide

Work through them in order the first time. Once you understand the whole picture, any lesson stands alone for teaching a specific topic.
